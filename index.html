Email Editor

Take a look: https://github.com/KuriousAgency/email-editor

Project information

A plugin to allow the use of CraftCMS Entry elements to build content for emails. System allows the linking of entries to system and commerce emails which can then be sent with content from the entry. Emails can be test sent from an entry page and test variables can be set up on each entry to allow for any variables required in a template such as an order detail email. Allows the creation of custom system messages and provides an easy way for plugins to add their own custom messages to be linked to an entry for content.

Personal Involvement

  • Initial build of version 1 of the plugin.
  • Rebuilding the plugin for version 2 and removing custom email elements to make use of Craft Entry elements and system messages.
